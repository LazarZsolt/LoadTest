This repo consist a Load tester application created by Zsolt Lázár

Ways of running the application: 
First way
	Open a command line in this folder:
		run: docker build -t loadtest .
	to run the application:
		run:docker run loadtest python /LoadTest.py --help

Second way
	Install python 2.x
	run: pip install pause
	run: python LoadTest.py --help


You are going to get the necessary information about the parameters there.
